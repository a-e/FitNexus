This suite contains demo tests for Rsel. In order to run them, you must install the Rsel development dependencies, then start the test servers:
{{{
$ cd submodules/rsel
$ bundle install
$ rake servers:start
}}}
Then run either the entire suite by clicking "Suite" on this page, or individual tests from the subpages listed below.
